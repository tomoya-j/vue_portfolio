```
npm install
```

```

```

### Compiles and minifies for production
```
npm run build
```

```

```

### 概要

vue.jsを使ってspaのプロフィールサイトを作ります。
環境構築はnpmとVue CLIを使い行います。

ルーティング設定とUIデザインはVuerouterとVuetifyを使って行います。
